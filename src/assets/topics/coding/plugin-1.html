雖然五年級……，但因為六年級參加校隊的關係，就此……，國一也因為……，就沒再……了。 直到國二接觸 Minecraft 這款遊戲後，一切才發生了變化。

Minecraft 就如同……一般，是一個可以自由……的遊戲， 不僅如此，……，在一次偶然中，我了解了原來你可以在自己的 Minecraft 伺服器裡面安裝所謂的「插件」 ，來讓……！這點對於……的我來說，實在很有吸引力，於是我立馬……也就這樣一腳踏進了插件這坑中。

程式的創作，來自需求

然而，隨著伺服器的玩家越來越多，我漸漸發現網路上的插件好像不太能滿足我的需求， 不是功能過多導致一件很簡單的事情要搞老半天，就是找不到能解決目前問題的插件。

這種情行經歷了一次、兩次、三次…後，我開始感到……，也衍生了想自己做的念頭。 就這樣，我開始學習插件、學習Java，學著自主開發插件。

粗糙的開始

一開始我根本不知道甚麼是Java，只知道他是插件使用的程式語言，也因為對這個東西不熟， 所以只能跟著Youtube上的教學影片，他做甚麼，我就跟著寫甚麼，這段程式碼是怎麼運作的、為甚麼……等， 都不是很清楚。因此，即使能照著影片做出成品，但只要一離開教學，我的腦袋就會跟全新的專案一樣：一片空白。

在……下，我終究還是買了一本 Java 的工具書來看，也因為有了她的幫忙， 我逐漸對……，也慢慢了解……，以及……。透過寫插件的過程，我將工具書上所學到的技巧一點一點地發揮在專案上， 在一次又一次的嘗試過程中，我也逐漸奠定了自己的程式語言基礎。
