曾經有段日子，我與另外一位朋友A……，但當時遇到了一個很煩人的問題： 為了程式碼能同步，只要A正在開發該插件，我就甚麼不能做，只能乾等他寫完後， 將所有文件壓縮成ZIP並丟到Google雲端後再去下載解壓縮，實在很浪費時間， 不過遇到這個問題，倒是讓我們發現了Git與Github這兩款很實用的版控工具！

有了它，我跟A都能在任何時候、任何地點自由開發與更改專案， 只要透過Git的Commit與Push功能，便能輕鬆同步我們兩人之間的變動，大幅增加了開發效率！

後來我才發現，原來Git還有能將文件回朔的 rollback 功能、額外分支的 branch 系統等...， 從那之後，除了團隊專案外，我自己的專案也一律會使用Git來做版本控制， 他也不負所望的於很多關鍵時刻救了我一命😅
