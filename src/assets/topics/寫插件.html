創造自己喜歡的遊戲，是許多人小時的夢想。

目前就讀高中三年級的林杰暉也是如此，熱愛玩 Minecraft 的他， 在前陣子因為瘋迷全球的吃雞遊戲：Pubg 而有了想將這兩種遊戲結合的衝動，於是著手開始製作魯大頭UHC。

林杰暉表示：「在一開始，我對程式語言根本一竅不通，是為了要製作這個東西，才會開始學 Java 及相關工具」， 在學習程式的途中，他不但慢慢增今自己對程式語言的了解，同時也學會了很多協助開發的工具， 包含 Maven、Gradle、Git、Github..等。

「而且說真的，我一開始也沒想到最後他會變成一個商品賣出去， 當初很單純只是想把它做出來，讓別人在 Minecraft 也能玩到高品質的遊戲而已 對於杰暉來說，賺錢並不是一開始的目的， 「只是後來在家人討論說椅子坐的不太舒服想換的時候，他們要我自己出一半， 所以就試試看把插件包裝一下，看能不能多少獲得一些收益來分擔支出」

最後，這份插件在銷售方面獲得了不錯的成績，杰暉也得以將這些收入拿去作為升級電腦環境的經費。 在採訪的最後，他說道：「我想，要是我當初是抱持著要賺錢的心情在寫這插件的話，或許到最後反而會甚麼都沒有」 對於能將販售插件賺錢，完全在他意料之外：「我從沒想過有天我真的能用插件賺錢，現在看來，真的是一段滿酷的旅程！」
